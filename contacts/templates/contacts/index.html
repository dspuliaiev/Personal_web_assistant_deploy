Welcome to Personal Assistant JARVIS5

Contact Management

The Personal Assistant project includes a comprehensive contact management system. It allows users to store contacts with names, addresses, phone numbers, emails, and birthdays in a contacts book. Users can easily add, edit, or delete contacts as needed.

Note Taking and Organization

Another crucial aspect of the Personal Assistant is its note-taking functionality. Users can store textual information as notes and organize them efficiently. Additionally, they can add tags or keywords to categorize notes based on topics or subjects. The system enables users to search for specific notes and provides sorting options based on tags for easier access.

Cloud File Management and News Summary

The project also offers cloud file management features, allowing users to upload files to a cloud service and access them from anywhere. It categorizes files into different types such as images, documents, and videos, and users can filter files based on categories.
